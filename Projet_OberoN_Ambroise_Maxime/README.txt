Nom du projet : OberoN

Description des éléments :
 - ./img : dossier contenant des images pour l'application.
 - ./jar : dossier contenant une version exécutable du projet avec un fichier d'exemple "exemple.ser" (le sous dossier ".cache" sert de cache au logiciel).
 - ./OberoN : sources '.java' du projet.
 - ./rapport.pdf
 - ./diagrammes : dossier contenant les diagrammes des classes.

N.B. : les fichiers de sauvegarde comportent l'extension .res.
